# CreditWise-ML: Loan Approval Prediction System

An end-to-end Machine Learning project that predicts loan approval status based on applicant data using multiple classification algorithms.

---

##  Features

* Complete ML pipeline implementation
* Data Cleaning & Exploratory Data Analysis (EDA)
* Feature Engineering & Encoding
* Model Training:

  * Logistic Regression
  * K-Nearest Neighbors (KNN)
  * Naive Bayes
* Model Evaluation:

  * Precision
  * Recall
  * F1 Score
* Comparative performance analysis

---

##  Problem Statement

Financial institutions need reliable systems to determine whether a loan should be approved.
This project builds a predictive model to assist in decision-making using historical data.

---

##  Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

---

## Models Used

* Logistic Regression (Baseline & Interpretable)
* KNN (Distance-based classification)
* Naive Bayes (Probabilistic approach)

---

##  Evaluation Metrics

* Precision
* Recall
* F1 Score

> Focused on balancing false positives and false negatives using F1-score.


##  Key Insights

* Income and credit history play a major role in loan approval
* Model performance varies depending on feature scaling and encoding
* Naive Bayes provided strong interpretability
